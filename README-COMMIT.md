## Commit Dabar:

**Jei naudojate GitHub Desktop:**
1. Atidarykite GitHub Desktop
2. Matysite visus pakeistus failus
3. Commit message: `Fix EADDRINUSE and local development - prevent multiple PM2 instances and fix localhost endpoint`
4. Spauskite "Commit to main"
5. Spauskite "Push origin"

**Jei naudojate Visual Studio Code:**
1. Atidarykite Source Control (Ctrl+Shift+G)
2. Matysite visus pakeistus failus
3. Spauskite "+" prie kiekvieno failo (stage)
4. Parašykite commit message: `Fix EADDRINUSE and local development - prevent multiple PM2 instances and fix localhost endpoint`
5. Spauskite "Commit"
6. Spauskite "Sync Changes" arba "Push"

**Jei git įdiegtas:**
